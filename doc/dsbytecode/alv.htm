ALV(dsByteCode::ebcALV,49) - Add Locale Variable
Description
Creates a new local variable with the given type and stores the result on the local variable stack.
Syntax
[byte]     dsByteCode::ebcALV
[dsClass*] type
Parameters
type
Specifies the type of the new local variable to create. Any class existing in the engine context is valid except void or null.
