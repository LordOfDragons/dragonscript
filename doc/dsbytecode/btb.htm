BTB(dsByteCode::ebcBTB,73) - Begin Try Block
Description
Marks the beginning of a new try-block. Try blocks protect code against unhandled exceptions if the JOE* commands are correctly set. Upon this Command the engine creates a new try-handling object and stores it ontop of the try-stack. All upcoming JOE* commands will be assigned to the topmost try-handler object. Thus specify first the BTB command and then all JOE* commands matching the exceptions to catch.
Syntax
[byte]     dsByteCode::ebcBTB
Parameters
none
