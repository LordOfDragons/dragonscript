CALL(dsByteCode::ebcCall,70) - Call Function
Description
Calls the function indicated by the index in a late-binding fashion popping the needed amount of arguments off the argument stack, building an argument list from them and passing the execution over to the fuction. The function executed is the most recent function definition found in the object class the function is called upon. Any function owned by one of the classes in the engine space can be used. If the function is a non-static function the execution object is taken as the object the function is called upon. If the function is a static function the execution object is ignored and null is passed to the function called.
Syntax
[byte] dsByteCode::ebcCall
[int]  function-index
Parameters
function-index
Function to call specified by an index into the function table of the appropriate object class. The index has to be in the range of 0 to the count of functions in the approtrate object class. To obtain a correct function index get the function list (GetFuncList) from the caller class, look for the function you need (FindBestFunction) and get the index for this function (GetIndexOf). This function has to be member of one of the classes existing in the engine space. Arguments stored for calling the function have to be stored in reverse order (argument is a stack). The function expects them to be in correct order upon execution.
