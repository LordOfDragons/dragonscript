CAST(dsByteCode::ebcCast,67) - Cast Operation
Description
Casts the execution object to the type specified if possible. If the cast is not possible an exception of type EInvalidCast will be thrown. If the execution object is of a primitive type (byte,bool,int,float) the value will be converted to the new type (with data loss if implied). If the execution object is of an object type only the type will be changed if the new type exists in the object class inheritance tree.
Syntax
[byte]     dsByteCode::ebcCast
[dsClass*] type
can throw EInvalidCast
Parameters
type
Specifies the type to cast to. Any class existing in the engine context is valid except void or null.
