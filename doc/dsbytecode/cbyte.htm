CBYTE(dsByteCode::ebcCByte,1) - Constant Byte Value
Description
Creates a new constant byte value and stores it as the current execution object.
Syntax
[byte] dsByteCode::ebcCByte
[byte] value
Parameters
value
The byte value to load.
