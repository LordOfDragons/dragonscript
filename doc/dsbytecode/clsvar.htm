CLSVAR(dsByteCode::ebcClsVar,10) - Load Class Variable
Description
Loads a class variable of the current execution object and stores it back into the execution object. The class variable must exists.
Syntax
[byte]        dsByteCode::ebcClsVar
[dsVariable*] variable
Parameters
variable
Specifies the pointer to the class variable to load on the current execution object. The variable specified here must exists in the class (either direct or in one of the base classes). Only variable pointers returned by one of the
dsVariable *dsClass::FindVariable(...)
functions are considered beeing safe to use here.
