ETB(dsByteCode::ebcETB,74) - End Try Block
Description
Marks the end of a try-block. Upon this command the topmost try-handler object is removed from the try-stack and freed. Use this command only after finishing all commands in the try block and not in the JOE* catch handlers. The engine already removes the topmost try-handler object after an exception has been caught or rethrown so calling ETB there can result in nasty exceptions or crashes.
Syntax
[byte]     dsByteCode::ebcETB
Parameters
none
