FLV(dsByteCode::ebcFLV,50) - Free Locale Variable
Description
Removes the given number of local variables from the local variable stack and frees them.
Syntax
[byte]  dsByteCode::ebcFLV
[short] count
Parameters
count
Specifies the number of local variables to remove from the local variable stack. All removed local variables are immediatly freed afterwards.
