JCEL(dsByteCode::ebcJCEL,63) - Jump If Case Equal (long)
Description
Moves the CP (Code Pointer) by an offset specified in int range (very long jumps) if the execution objects is equal to the case value specified. The maximal jump offset is in the range of -2'147'483'648 to 2'147'483'647 bytes. The jump will be carried out (if the condition is satisfied) after the offset has been read from the ByteCode stream and the CP has been updated.
Syntax
[byte] dsByteCode::ebcJCEL
[int]  case-value
[int]  offset
Parameters
case-value
The case value to compare the execution object with. This value has to be an integer value.
offset
Offset to move the CP. Make sure to read this value as signed int. The range for this offset value is -2'147'483'648 to 2'147'483'647.
