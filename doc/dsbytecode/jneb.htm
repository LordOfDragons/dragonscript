JNEB(dsByteCode::ebcJNEB,58) - Jump If Not Equal (byte)
Description
Moves the CP (Code Pointer) by an offset specified in byte range (very short jumps) if the execution objects is false. The maximal jump offset is in the range of -128 to 127 bytes. The jump will be carried out (if the condition is satisfied) after the offset has been read from the ByteCode stream and the CP has been updated.
Syntax
[byte] dsByteCode::ebcJNEB
[char] offset
Parameters
offset
Offset to move the CP. Make sure to read this value as signed char. The range for this offset value is -128 to 127.
