JNES(dsByteCode::ebcJNES,59) - Jump If Not Equal (short)
Description
Moves the CP (Code Pointer) by an offset specified in short range (common jumps) if the execution objects is false. The maximal jump offset is in the range of -32'768 to 32'767 bytes. The jump will be carried out (if the condition is satisfied) after the offset has been read from the ByteCode stream and the CP has been updated.
Syntax
[byte]  dsByteCode::ebcJNES
[short] offset
Parameters
offset
Offset to move the CP. Make sure to read this value as signed short. The range for this offset value is -32'768 to 32'767.
