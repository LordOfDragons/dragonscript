JOEL(dsByteCode::ebcJOEL,66) - Jump On Exception (long)
Description
Registers an entry-point to move the CP (Code Pointer) to marked by an offset specified in int range (very long jumps). Moves the CP (Code Pointer) to the point marked if later an exception occurs having the type specified. The maximal jump offset is in the range of -2'147'483'648 to 2'147'483'647 bytes. The entry-point will be calculated based on the location after reading the offset from the ByteCode stream and updating the CP plus this offset.
Syntax
[byte]     dsByteCode::ebcJOEL
[dsClass*] exception-type
[int]      offset
Parameters
exception-type
The type of the exception to be thrown for the jump to be triggered. The exception can occur any time after registring the entry-point.
offset
Offset to mark the entry-point. Make sure to read this value as signed int. The range for this offset value is -2'147'483'648 to 2'147'483'647.
