LOCVAR(dsByteCode::ebcLocVar,12) - Load Locale Variable
Description
Loads a local function variable and stores it into the execution object.
Syntax
[byte]  dsByteCode::ebcLocVar
[short] index
Parameters
index
Local Variables are stored on a stack. Thus the last local variable created has to be refered to using the index 0. The Compiler manages an internal stack to keep track of the correct local variable index numbers at each time. If you do this by hand be carefull to pick the correct number. To find the correct index of a local variable the function
int dspParserCheckCode::FindLocalVariable(const char *Name, bool OnlyLocal) const
has to be used.
