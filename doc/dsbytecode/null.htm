NULL(dsByteCode::ebcNull,7) - Null Object
Description
Creates a new result with a null value and stores it as the current execution object.
Syntax
[byte]     dsByteCode::ebcNull
[dsClass*] type
Parameters
type
Pointer to a dsClass specifying the type of the null object. Any null object created must be of a type having DSTP_OBJECT as primtive type. You can later on cast this null object to any other type as long as the value stays null. Otherwise a cast exception can occur.
