OPASS(dsByteCode::ebcOpAss,36) - Assign Operation
Description
Pops one argument off the argument stack and assigns that value to the current execution object. The execution object stays the same. This operator can only be applied to objects of the following types: byte, bool, int, float, objects. In all cases the result will be of the same type as before the assignment. In all cases the type of the argument from the stack has to be same as the type of the execution object. If the execution object is of a primitive type (byte,bool,int,float) the value of the argument will be copied. If the execution object is of an object type the reference to the real object is copied and stored. The Engine takes care of freeing no more referenced objects itself (no garbage collector needed).
Syntax
[byte] dsByteCode::ebcOpAss
Parameters
none
