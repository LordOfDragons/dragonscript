OPDIV(dsByteCode::ebcOpDiv,21) - Primitive Division Operation
Description
Pops one argument off the argument stack, divides the current execution object through this argument and stores the result as new execution object. This operator can only be applied to objects of the following types: byte, int, float. If the execution object is of type byte the new type will be int. If the execution object is of type int or float the type will be the same. In all cases the type of the argument from the stack has to be same as the type of the execution object. If the argument from the stack equals 0 an exception of type EDivisionByZero will be thrown.
Syntax
[byte] dsByteCode::ebcOpDiv
can throw EDivisionByZero
Parameters
none
