OPDIVA(dsByteCode::ebcOpDivA,38) - Primitive Division Assign Operation
Description
Pops one argument off the argument stack, divides the current execution object through this value and assigns the result to it. The execution object stays the same. This operator can only be applied to objects of the following types: byte, int, float. In all cases the result will be of the same type as before the assignment. In all cases the type of the argument from the stack has to be same as the type of the execution object. If the argument from the stack equals 0 an exception of type EDivisionByZero will be thrown.
Syntax
[byte] dsByteCode::ebcOpDivA
can throw EDivisionByZero
Parameters
none
