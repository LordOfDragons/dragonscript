OPEQ(dsByteCode::ebcOpEq,31) - Primitive Equal Comparisson
Description
Pops one argument off the argument stack, compares if the current execution object is equal to the argument and stores the result as new execution object. This operator can only be applied to objects of the following types: byte, int, float. In all cases the result will be of type bool containing the result of the comparisson. In all cases the type of the argument from the stack has to be same as the type of the execution object.
Syntax
[byte] dsByteCode::ebcOpEq
Parameters
none
