OPINC(dsByteCode::ebcOpInc,15) - Primitive Increment Operation
Description
Takes the current execution object and increments its value by one. The current execution object stays the same. This operator can only be applied to objects of the following types: byte, int, float
Syntax
[byte] dsByteCode::ebcOpInc
Parameters
none
