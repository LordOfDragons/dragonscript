OPINV(dsByteCode::ebcOpInv,19) - Primitive Invert Operation
Description
Takes the bitwise inverse of the current execution object and stores this value as the new execution object. This operator can only be applied to objects of the following types: byte, int. In both cases the new value will be of type int.
Syntax
[byte] dsByteCode::ebcOpInv
Parameters
none
