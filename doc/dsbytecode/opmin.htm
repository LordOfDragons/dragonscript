OPMIN(dsByteCode::ebcOpMin,17) - Primitive Minus Operation
Description
Takes the negative value of the current execution object and stores this value as the new execution object. This operator can only be applied to objects of the following types: byte, int, float. If the execution object is of type byte the result will be int. If the execution object is of type int or float the result will be of the same type.
Syntax
[byte] dsByteCode::ebcOpMin
Parameters
none
