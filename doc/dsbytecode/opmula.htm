OPMULA(dsByteCode::ebcOpMulA,37) - Primitive Multiply Assign Operation
Description
Pops one argument off the argument stack, multiplies the current execution object with this value and assigns the result to it. The execution object stays the same. This operator can only be applied to objects of the following types: byte, int, float. In all cases the result will be of the same type as before the assignment. In all cases the type of the argument from the stack has to be same as the type of the execution object.
Syntax
[byte] dsByteCode::ebcOpMulA
Parameters
none
