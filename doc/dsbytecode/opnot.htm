OPNOT(dsByteCode::ebcOpNot,18) - Primitive Not Operation
Description
Takes the logical negation of the current execution object and stores this value as the new execution object. This operator can only be applied to objects of the following types: byte, bool, int, float. If the execution object is of type bool the result will be the negation of the current value and the type stays the same. If the execution object is of type byte, int or float the result will be false for a value equal 0 and true otherwise.
Syntax
[byte] dsByteCode::ebcOpNot
Parameters
none
