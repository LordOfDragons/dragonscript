OPRS(dsByteCode::ebcOpRS,26) - Primitive Right Shift Operation
Description
Pops one argument off the argument stack, bitwise shifts the current execution object to the right side and stores the result as new execution object. This operator can only be applied to objects of the following types: byte, int. If the execution object is of type byte the new type will be int. If the execution object is of type int the type will be the same. In all cases the type of the argument from the stack has to be same as the type of the execution object. Any bits shifted outside the 0th bit will be omitted and not rotated back in on the other end. New bits shifted in are 0.
Syntax
[byte] dsByteCode::ebcOpRS
Parameters
none
