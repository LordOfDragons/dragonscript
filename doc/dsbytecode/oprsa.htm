OPRSA(dsByteCode::ebcRSA,43) - Primitive Right Shift Assign Operation
Description
Pops one argument off the argument stack, bitwise shifts the current execution object to the right side and assigns the result to it. This operator can only be applied to objects of the following types: byte, int. In all cases the result will be of the same type as before the assignment. In all cases the type of the argument from the stack has to be same as the type of the execution object. Any bits shifted outside the 0th bit will be omitted and not rotated back in on the other end. New bits shifted in are 0.
Syntax
[byte] dsByteCode::ebcOpRSA
Parameters
none
