OPXOR(dsByteCode::ebcOpXor,35) - Primitive Xor Operation
Description
Pops one argument off the argument stack, carries out a bitwise XOR of the current execution object with the argument and stores the result as new execution object. This operator can only be applied to objects of the following types: byte, int. In both cases the result will be of type int. The operation checks all bits of both values and sets a bit to 1 if at the source or destination bit are 1 but not both.
source bitdestination bitresult bit
000

110
In all cases the type of the argument from the stack has to be same as the type of the execution object.
