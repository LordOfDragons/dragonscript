PARAM(dsByteCode::ebcParam,11) - Load Function Parameter
Description
Loads an argument passed to the current function in process and stores it into the execution object. The argument index has to be in the range defined by the function in process.
Syntax
[byte] dsByteCode::ebcParam
[byte] index
Parameters
index
Specifies the index of the argument to load. The Index is in the range from 0 to Count-1 with Count beeing the maximum number of arguments specified by the function definition. The first argument is the index 0. Thus if a function has three arguments defined:
MyFunction(int arg1, byte arg2, int arg3)
arg1 is the argument with index 0 and arg3 the last argument with index 2.
A function can only have a maximum of 256 arguments.
