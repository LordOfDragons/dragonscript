PDEC(dsByteCode::ebcPDec,48) - Post Decrement Operation
Description
Creates a copy of the current value of the execution object, decrements the execution object and stores the copy as new execution object. This operator can only be applied to objects of the following types: byte, int, float.
Syntax
[byte] dsByteCode::ebcPDec
Parameters
none
