PINC(dsByteCode::ebcPInc,47) - Post Increment Operation
Description
Creates a copy of the current value of the execution object, increments the execution object and stores the copy as new execution object. This operator can only be applied to objects of the following types: byte, int, float.
Syntax
[byte] dsByteCode::ebcPInc
Parameters
none
