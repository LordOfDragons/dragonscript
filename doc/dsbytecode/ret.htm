Main / RET(dsByteCode::ebcRet,51)
RET(dsByteCode::ebcRet,51) - Return
Description
Stops the execution of the current function and exits. Every function having has to own such a command for all control path possible. Failing to do so will result in an exception or crash (depending if __DEBUG__ is specified).
Syntax
[byte]  dsByteCode::ebcRet
Parameters
none
