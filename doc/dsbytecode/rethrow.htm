THROW(dsByteCode::ebcThrow,75) - throws an exception
Description
Rethrows the last exception thrown and caught by a JOE* command. This command can only be used inside a JOE* handler block and not inside or outside a try-block. The current try-handler will no more take action thus any other JOE* handlers in this try-handler will not be taken into account whilest thrown the exception. If a corresponding JOE* command outside this try-handler has been invoked earlier the given JOE* handler will be fired. Otherwise the engine halts the execution of the code and outputs an exception trace.
Syntax
[byte]     dsByteCode::ebcReThrow
Parameters
none
