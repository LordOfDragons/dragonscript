THIS(dsByteCode::ebcThis,8) - Load This Object
Description
Stores the object on which the current function in process has been called on as the current execution object. This command must only be used in a non-static function (otherwise a null-pointer exception occurs).
Syntax
[byte] dsByteCode::ebcThis
Parameters
none
