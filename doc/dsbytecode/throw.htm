THROW(dsByteCode::ebcThrow,75) - throws an exception
Description
Throws the execution object as an exception. The execution object has to castable to Exception. If a corresponding JOE* command has been invoked earlier the given JOE* handler will be fired. Otherwise the engine halts the execution of the code and outputs an exception trace.
Syntax
[byte]     dsByteCode::ebcThrow
Parameters
none
