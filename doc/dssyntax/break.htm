Break Statement
Description
The Break Statement immediatly quits the inner most For, While or Select statement enclosing the statement-block the Break Statement is in.
Syntax
break
Parameters
none
Example
This example loops as long no key has been pressed. Keyboard is a native class of some fictive application. 
while true // loops indefinitly
   if Keyboard.GetAnyKeyPressed()
      break
   end
   // do some work here each time a key is not pressed
end
