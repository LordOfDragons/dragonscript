Main / Control Flow / Continue Statement
Break Statement
Description
The Continue Statement causes the innermost For or While Statement to skip over the code following the Continue Statement and to check the conditions for another loop. In a For Statement this causes the counter variable to be increased before the check is done. In a While Statement the condition is directly checked after invokation.
Syntax
continue
Parameters
none
Example
In this example we count from 0 to 10 and sum up the values except the value 5. If we get there we skip the rest of the code and go for the next round. 
for i = 0 to 10 do{
   if i == 5 then continue;
   vResult += i;
}
