Control Flow
Description
Control Flow statements allow scripts to execute different code depending on conditions or execute a chunk of code multiple times. On one side there are the Conditional-Branch statements forking code execution depending on a condition. On the other side there are the Loop-Constructs executing code indefinitly until a certain condition is fulfilled.
Content
If-Else Statement
Inline If-Else Expression ( ? : )
Select Statement
For Statement
While Statement
Break Statement
Continue Statement
