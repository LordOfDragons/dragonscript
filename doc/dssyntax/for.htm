For Statement
Description
The For Statement executes a statement for a given number of times using a counter variable. The counter variable is incremented/decremented by 1 (or another value) each time the statement is executed. As soon as the counter variable reaches the limit the For Statement exits.
Syntax
for counter-variable = start-value to end-value [ step step-value ] do
   statement-block
	  
for counter-variable = start-value downto end-value [ step step-value ] do
   statement-block
Parameters
counter-variable
The variable used for counting. This variable has to be declared already (thus for does not declare this variable). The type of the variable has to be int.
start-value
Defines the value to start counting with.
end-value
Defines the value at to stop counting. The For Statement exits as soon as the counter variable reaches this value. For the up version of For this value has to be greater than the start-value. For the downto version of For this value has to be smaller than the start-value. Failing to do so can result in an infinite loop (dead loop).
step-value
Defines the value to be added to/subtracted from the counter value each time the statement has been executed. For both versions of For this value has to be positive. In the case of the up version the step value is added to the counter value. In the case of the downto version the step value is subtracted from the counter value. Failing to do so can result in an infinite loop (dead loop). If no step value is specified the default step value is used which is 1.
statement-block
The statement to execute each time the loop is run.
Example
This example adds 5 times the value of vOp1 to result. The local variable i is already defined as int. As soon as the values becomes 5 the loop is left so 5 is no more added to the result. 
for i = 0 to 5 do
   vResult += vOp2;
This example counts down from 5 to 0 and prints each time the value to the console. As soon as the values becomes 0 the loop is left so for 0 nothing is printed out. 
for i = 5 downto 0 do{
    System.Print("Seconds left: " + i);
}
Here we sum up all values starting at 3 and incrementing each time by 2 until we reach 11. Again at 11 the loop ends and 11 is no more added to the result. 
for i = 3 to 11 step 2 do
   vResult += vOp2;
