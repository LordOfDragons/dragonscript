If-Else Statement
Description
The If-Else Construct executes different code blocks depending on a condition. If the condition specified in the Statement turns out to be true then the If-Statement-Block is executed. Otherwiese the Else-Statement-Block is executed if present and if not the If-Statement-Block is just skipped.
Syntax
if condtion then
   if-statement-block
[ else
   else-statement-block ]
Parameters
condition
Is an expression beeing of type or castable to bool.
if-statement-block
This statement is executed in the case the condition equals true otherwise it is skipped.
else-statement-block
This statement is executed in the case the condition equals false otherwise it is skipped. If not specifed the if-statement-block is just skipped in this case.
Example
Simple If-Construct executing the code only if a condition is satisfied otherwise it is skipped. 
if vOperator2 != 0 then
   vResult = vOperator1 / vOperator2;
The same situation but this time if the divider is 0 we execute another code branch. 
if vOperator2 != 0 then
   vResult = vOperator1 / vOperator2;
else
   vResult = 0;
If-Else Constructs can be nested to any deep. Just use an If-Else-Statement in the else clause. This example conducts different operations based on an input value. 
if vOperator = '+' then
   vResult = vOp1 + vOp2;
else if vOperator = '-' then
   vResult = vOp1 - vOp2;
else if vOperator = '*' then
   vResult = vOp1 * vOp2;
else
   vResult = 0;
Statements can also be statement blocks. This are the first two examples using statement blocks this time. 
if vOperator2 != 0 then{
   vResult = vOperator1 / vOperator2;
}
if vOperator2 != 0 then{
   vResult = vOperator1 / vOperator2;
}else{
   vResult = 0;
}
