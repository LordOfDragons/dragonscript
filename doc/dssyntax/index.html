DragonScript - Syntax
Introduction
Welcome to this documentation over the DragonScript Syntax.
It will give you a completly reference over the entire language standard of this scripting language. The documentation is held in simple HTML so you don't need special browser. Any style sheet compliant browser is able to display this documentation properly.
