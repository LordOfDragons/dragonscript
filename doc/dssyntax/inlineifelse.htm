Inline If-Else Expression ( ? : )
Description
The Inline If-Else Expression is a special version of the If-Else-Statement. It is called Inline because it is an expression not a statement and can thus be used as part of an expression. If the condition specified in the expression turns out to be true then the If-Expression-Block is executed otherwise the Else-Expression-Block.
Syntax
condtion ? if-expression-block : else-statement-block
Parameters
condition
Is an expression beeing of type or castable to bool.
if-expression-block
This expression is evaluated in the case the condition equals true.
else-expression-block
This expression is evaluated in the case the condition equals false.
Example
Inline If-Else-Expressions are usually used to replace bulky If-Else-Statements which are simple enough to be squeezed on a one-liner. The first statement is though the same as the second one following it. 
// expression version
vResult = vOp2 != 0 ? vOp1 / vOp2 : 0;
// statement version
if vOp2 != 0 then{
   vResult = vOp1 / vOp2;
}else{
   vResult = 0;
}
