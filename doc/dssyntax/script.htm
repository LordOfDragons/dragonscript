Script and Compilation Settings
Description
Script and Compilation Settings affect the way how a compiler or interpreter of the DragonScript language processes the scripts. All those settings heavily depend on the host application used and will differ from application to application. The documentation describes those settings in general. Please refer to the documentation coming with the application you want to write scripts for to find out what settings the application uses in particular.
