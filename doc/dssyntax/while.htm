While Statement
Description
The While Statement executes a statement as long as a a condition is true. There are two versions of while. The first version checks the condition before the statement is executed for the first time. The second version executes first the statement and checks then for the first time the condition.
Syntax
while condition do
   statement-block
	  
do
   statement-block
while condition
Parameters
condition
The While Statement executes the statement as long as this expression is true. The condition has to be of or castable to type bool.
statement-block
The statement to execute each time the loop is run.
Example
This example loops as long the key has not been pressed. Keyboard is a native class of some fictive application. 
while Keyboard.GetKeyPressed(Keyboard.KEY_ESCAPE) do{
   // do some work here each time the key is not pressed
}
This time we want to add a character to a string and check each time if the new character added is a terminator character. Because we want to add the char before we check it we use the second While version here. vString is a local string variable initialized to "" and vChar a local byte variable. 
do{
   vChar = Keyboard.GetNextChar();
   vString += vChar;
}while vChar != Keyboard.KEY_RETURN;
